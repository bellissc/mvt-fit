# MVT Fitness Demo Guide

This short guide is for anyone testing the MVT Fitness prototype in a browser.

## Demo Accounts

- Trainer login: `coach@coachflow.test`
- Trainer password: `demo123`
- Client login: `alex@coachflow.test`
- Client password: `demo123`

## Before You Start

- Open the demo link in a browser
- If anything looks off, click `Reset demo data` in the top bar
- The app stores demo changes in your own browser only

## Trainer Flow

1. Sign in with the trainer account.
2. Review the client roster on the main trainer screen.
3. Click `Manage` beside Alex to open the client detail page.
4. Review:
   - profile details
   - current plan
   - recent progress
5. In the workout area:
   - search exercises
   - choose a date
   - set sets, reps, and rest
   - add exercises to the client routine
   - click `Submit workout`
6. In the meal plan area:
   - choose a week
   - choose a day and meal
   - search foods or use quick-add foods
   - build the weekly meal plan
   - click `Submit meal plan`
7. In the admin area:
   - set subscription tier and contract length
   - save subscription settings
   - test `Add credit card`
   - test `Send secure card link`
8. In the exercise media area:
   - choose an exercise
   - add a demo video URL
   - add a thumbnail label
   - save the media setup

## Client Flow

1. Sign out.
2. Sign in with the client account.
3. Review:
   - notifications
   - assigned workout
   - assigned meal plan
   - recent logs
4. In the workout area:
   - open exercise demo media
   - mark workout items complete
5. In the progress area:
   - enter body weight
   - choose how energy felt
   - add notes or a video link
   - save the log
6. Check the recent progress chart and history items.

## Notes For Testers

- This is a prototype for UX testing only.
- Billing buttons are demo actions and do not charge a real card.
- Notifications are simulated inside the app.
- Data is not shared between different people using the demo link.

## If Something Breaks

1. Refresh the page.
2. Click `Reset demo data`.
3. Log in again.
